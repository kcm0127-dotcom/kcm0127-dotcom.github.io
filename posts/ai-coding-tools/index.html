Claude Code (Anthropic)

장점: 코드 품질 최상, 리팩토링·아키텍처 설계에 강함, 터미널 통합 자연스러움

단점: 구독제 비용 ($10~25/월), 긴 세션에서 컨텍스트 손실

OpenAI Codex CLI

장점: OpenAI 생태계 완벽 통합, GPT-5 기반 강력한 추론

단점: 양자화 모델 품질 저하, 종량제 과금 예측 어려움
